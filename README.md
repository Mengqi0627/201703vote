# 全栈投票项目开发环境
投票项目的开发环境，请参考《投票系统需求文档PRD》和《API》文档进行开发

## 运行项目

克隆项目

```
> $ git clone https://git.coding.net/canfoo/vote-development.git
```
在vote目录下安装依赖包

```
> $ npm install
```
启动服务器

```
> $ npm run start
```

## 项目流程
- 老板的一个想法或创意
- 产品经理进行原型设计 Axure sketch => 原型图 需求文档
- UI进行设计 => PSD
- 前端开发人员 => HTML+交互
- 后端交互
- 测试
- 上线
